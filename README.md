# KM-MiniProgram
简单的学习了一下微信小程序，看来一下官网，想写一个自己的微信小程序，在线上跑跑，于是觉得开始开发一个备忘录的小程序；<br>
因为这个相对简单，主要采用的是本地存储，官方提供的API ‘wx.setStorage’，没有使用后端接口进行数据的存储和获取；<br>
项目名称为：KM备忘录；<br>
开发时间：大概是2018.3.28--2018.4.10的下班空闲时间；<br>
开发过程：手画原型图、流程图，无设计稿，开干！！！ <br>
